Modern full-stack ecommerce platform built using the MERN stack.

🚀 Features
-User Authentication (JWT)
-Email OTP Verification
-Product Listings
-Product Categories
-Product Details Page
-Add to Cart
-Wishlist Management
-Checkout System
-Order Management
-Product Reviews
-Responsive Modern UI
-Protected Routes
-REST API Integration

🛠️ Tech Stack
-Frontend
-React.js
-Vite
-Tailwind CSS
-React Router DOM
-Axios
-React Hot Toast
-Swiper.js
-Backend
-Node.js
-Express.js
-MongoDB
-Mongoose
-JWT Authentication
-Nodemailer

⚙️ Installation
Clone Repository
git clone https://github.com/kishorrjeyachandran/Cartify.git

Frontend Setup
cd frontend
npm install
npm run dev

Backend Setup
cd backend
npm install
npm run dev


📦 Deployment
Frontend deployed on Vercel
Backend deployed on Render
Database hosted on MongoDB Atlas
